# OSQZ Security Portfolio

개인 보안 포트폴리오용 싱글 페이지 사이트 저장소입니다.  
GitHub Pages를 통해 아래 주소로 배포됩니다.

> https://o-sqz.github.io/portfolio/

---

## 1. 목적

- 정보보안 직무 지원 시 제출할 **웹 포트폴리오 페이지**
- 웹 보안 / 보안 자동화 / 사회공학기법 관련 **프로젝트 및 활동 정리**
- Dreamhack Wargame, Tistory 블로그, GitHub 등의 **흩어진 기록을 한 곳에서 정리**

---

## 2. 제작 환경 및 소요 시간

이 저장소는 다음 도구들을 활용해 **약 5시간 내에 초기 버전을 완성**했습니다.

- **ChatGPT** – 레이아웃 설계, HTML/CSS/JS 코드 초안, 구조 개선
- **Visual Studio Code** – 코드 편집 및 Git 연동
- **GitHub Web & GitHub Pages** – 저장소 관리 및 정적 페이지 호스팅
